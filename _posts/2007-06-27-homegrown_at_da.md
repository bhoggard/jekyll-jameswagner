<p><br /><br />
<a href="http://bloggy.com/mt/archives/004751.html"><br />
Renee Riccardo</a> is the curator of a group show, "Homegrown", currently at <a href="http://www.artcal.net/event/view/1/4889">David Krut Projects</a>, which draws artists and collaboratives from five corners of the country.  The threads running through the project include thread, itself in the case of several pieces, and a number of other <em>homegrown</em> materials and practices including tape, plastics, ribbon, foam, shells, refashioned found objects, glitter and collage.</p>

<p>There's much fun to be found here, but it's not all as playful as the materials might suggest.</p>

<p>The artists represented are <a href="http://www.scott-andresen.com/">Scott Andresen</a>, <a href="http://www.undo.net/cgi-bin/undo/pressrelease/pressrelease.pl?id=1173884855">Karen Azoulay</a>, <a href="/2005/07/_ive_had_this_w.html">Bethany Bristow</a>, <a href="/2003/07/michael_exposed.html">Orly Cogan</a>, <a href="http://www.allstonskirt.com/artists/dash/dashnew.html">Robin Dash</a>, <a href="http://www.misakikawai.com/">Misaki Kawai</a>, <a href="http://www.miamiartexchange.com/miami_art_articles/miami_art_articles_2007/unpacking_kuhl_leyton.html">Kuhl & Leyton</a>, <a href="http://www.greglamarche.com/">Greg Lamarche</a>, <a href="http://www.galerieperrotin.com/artiste_.php?id_=114&&nom_=Cristina%20Lei%20Rodriguez&&dossier=Cristina_Lei_Rodriguez">Cristina Lei Rodriguez</a>, <a href="http://www.artnet.com/Galleries/Artists_detail.asp?G=&gid=424019546&which=&aid=424177185&ViewArtistBy=online&rta=http://www.artnet.com/artist/424177185/margaret-lee.html">Margaret Lee</a>, <a href="http://bloggy.com/mt/archives/006365.html">LoVid</a>, <a href="http://www.mixedgreens.com/ArtWeb/html/artistbio.asp?artnum=45">Adia Millett</a>, <a href="http://bloggy.com/mt/archives/006362.html">Doug Morris</a>, <a href="https://flickr.com/photos/annefrancoise/">Anne-Francoise Potterat</a>, <a href="http://mocoloco.com/art/archives/001426.ph">Jon Rosenbaum</a>, <a href="http://www.monyarowegallery.com/somogyi/index.html">Erika Somogyi</a>, <a href="http://www.fallonandrosof.com/2005/09/lunching-and-quilting-with-lee.html">Lee Tusman</a>, and <a href="http://http://animalnewyork.com/2006/08/real_life_web_slinger_entangle_1.php">Jasmine Zimmerman</a>.</p>
